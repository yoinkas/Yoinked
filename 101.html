Starting Point

101

If you found this page, you are probably wondering where to begin. If you are unsure how to start your cybersecurity career, this is where we slow it down with simple exercises and quizzes so you can build momentum and get back into the flow.

command line exercises

Receiving files and traveling through directories

These are basic terminal habits you will use constantly. The goal is to get comfortable moving around, finding files, and pulling files onto your machine without guessing.

Exercise 1: Know where you are

Start by checking your current directory, then list what is inside it.

pwd
ls
ls -la

Try to explain the difference between ls and ls -la before moving on.

Exercise 2: Move through directories

Practice moving into folders, backing out, and jumping straight home.

cd Downloads
cd ..
cd ~

Your goal is to move around without getting lost.

Exercise 3: Create and organize

Make a practice folder, create a file, and move it into a new location.

mkdir cyber-lab
cd cyber-lab
touch notes.txt
mkdir logs
mv notes.txt logs/

Exercise 4: Receive a file

Get used to downloading or copying a file so you can inspect it locally.

wget https://example.com/file.txt
curl -O https://example.com/file.txt
cp /path/to/file.txt .

The lesson here is simple: receive the file first, then inspect it in a controlled place.

quizzes

Beginner-friendly questions

Use these to check whether the basics are starting to stick. Try answering them before revealing the answers.

Quiz 1

Question: What command shows you your current working directory?

Answer: pwd

Quiz 2

Question: What does FTP do wrong from a security perspective?

Answer: It sends credentials and traffic in plaintext.

Quiz 3

Question: What command is commonly used to list files in a directory on Linux?

Answer: ls

Quiz 4

Question: Why is a password like password123 weak?

Answer: It is common, predictable, and easy to guess with a dictionary attack.

matching game

Match the term to the definition

Read the terms on the left and match them to the correct meaning. This is a simple memory drill for beginner cyber vocabulary.

Directory

A folder used to organize files.

FTP

A file transfer protocol that is insecure because it does not encrypt traffic by default.

Packet Capture

A recorded set of network traffic that can be reviewed in tools like Wireshark.

Brute Force

A method of trying many passwords or values until one works.

Web Shell

A malicious script uploaded to a web server to execute commands remotely.

Privilege Escalation

The act of gaining higher-level permissions after initial access.
